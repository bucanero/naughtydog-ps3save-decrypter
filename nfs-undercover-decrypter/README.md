# nfs-undercover-decrypter

A tool to decrypt Need For Speed: Undercover PS3 save-games.

```
USAGE: ./nfs-undercover-decrypter [option] filename

OPTIONS        Explanation:
 -d            Decrypt File
 -e            Encrypt File
```
