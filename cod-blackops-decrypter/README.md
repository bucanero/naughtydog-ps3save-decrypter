# cod-blackops-decrypter

A tool to decrypt Call of Duty: Black Ops/Black Ops 2 PS3 save-games

```
USAGE: ./cod-blackops-decrypter [option] filename

OPTIONS        Explanation:
 -d            Decrypt File
 -e            Encrypt File
```
