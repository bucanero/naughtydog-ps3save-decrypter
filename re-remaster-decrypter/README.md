# re-remaster-decrypter

A tool to decrypt Resident Evil: HD Remaster PS3 save-games.

```
USAGE: ./re-remaster-decrypter [option] filename

OPTIONS        Explanation:
 -d            Decrypt File
 -e            Encrypt File
```
